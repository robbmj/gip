inOrder() → {CollectionGenerator}

The inOrder method is a generator that visits each node in the tree. The In Order algorithum first visits the root node's left subtree, then the root its self, then the right subtree.

The method is available in ES 2015 only, to perform an in order traversal in < ES 2015 use BinaryTree#inOrderTraversal instead.

Since:
  • 0.1.0
Source:
  • collections/trees/binarytree.interface.js, line 178
Returns:
Type
CollectionGenerator
Example
tree.addAll(2,1,3);
for (let e of tree.inOrder()) console.log(e); // 1,2,3
